# Welcome to pub-blog

This is the first post on our new Astro-powered automated news aggregator, migrated from the Jekyll-based feedmeup project.

## What's New

- Built with Astro 5 for better performance
- Maintains all existing features:
  - 40+ RSS/Atom feed sources
  - Keyword filtering and categorization
  - AI-powered summarization with Google Gemini
  - Automated weekly execution
